# AI Job Market Analysis

## Project Overview
Analysis of AI/ML job market trends using a dataset of 2,000 job postings. This project explores salary distributions, experience level patterns, and industry variations.

**Course:** Data 200: Data Systems for Data Analytics (Fall 2025)  

## Files
- `PROJECT 1 - AI Job Market.ipynb.ipynb` - Main analysis notebook
- `ai_job_market.csv` - Dataset (2,000 job postings)

## Key Findings
- **Salary Patterns**: E-commerce and Automotive industries offer highest average salaries
- **Experience Distribution**: Balanced hiring across Entry (702), Mid (668), and Senior (630) levels
- **Specialization Premium**: NLP/ML Engineering roles command highest median salaries
- **Weak Correlation**: Experience level alone explains little salary variation (r = 0.098)
